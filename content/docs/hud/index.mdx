## MT HUD Remastered
Minimalistic but complete FiveM HUD system

# Features
- Dynamic status HUD
- Car HUD
- Eletric Car HUD
- Bicycles HUD
- Aircrafts HUD
- Advanced Compass
- Simple notifications

**None of the Features are optional, it comes all together and the minimap hides on foot and Bicycles like in the preview video.**

# Dependencies:
- ox_lib;
- qb-smallresources;
- qbx_seatbelt;
- jg-stressaddon (optional);
- jg-mileage (optional);
