# MT Mining
Simple and interactive mining/mine script for FiveM

**Features done**
- Different custom rock props with different minerals for each rock
- Interactive animations with particles and sound effects
- Custom props (not included, check dependicies)
- Wash tub to wash minerals
- Furnance to melt minerals
- Customizable reward drops
- Custom sledge hammer weapon
- Buy and sell items on the integrated shops system

**Dependicies**
- ox_lib
- ox_target
- ox_inventory
- Renewed-Lib
- [Mining props by BZZ](https://bzzz.tebex.io/package/5113812)
