## Sledge hammer weapon
Add the following weapon to your ox_inventory/data/weapons.lua
```
['WEAPON_SLEDGEHAMMER'] = {
    label = 'Sledge Hammer',
    weight = 1200,
    durability = 0.1,
},
```
