# Learn how to install our Restaurants v3

## Inventory Items
This are the items to add in the `ox_inventory\data\items.lua`

**Required items**
```lua copy
["restaurant_food"] = {
    label = "Restaurant food",
    useable = true,
    weight = 400,
    stack = true,
    close = true,
    client = {
        export = 'mt_restaurants.useFood'
    },
    degrade = 1440 -- 24 hours
},
["restaurant_box"] = {
    label = "Food box",
    useLabel = 'Abrir',
    useable = true,
    weight = 200,
    stack = true,
    close = true
},
```

```lua copy
['k8kmkccdhq4bghqi.public.blob.vercel-storage.com'] = true,
['upload.mt-scripts.com'] = true,
```
