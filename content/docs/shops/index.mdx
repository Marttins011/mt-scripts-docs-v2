# MT Shops
Complete FiveM shops script with a clean UI

**Features**
- Clean, minimal Svelte UI
- Custom payment methods including Society
- Vending Machines
- Job, grade and license locked items
- Sell to stock based shops
- Sell items option
- Exports for external use with ease
- Optimized and secure against expolits and cheaters

**Dependicies**
- ox_target
- ox_lib
- ox_inventory
